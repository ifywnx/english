Câu điều kiện từ cơ bản đến nâng cao

Module này bao gồm zero, first, second, third và mixed conditionals. Đây là một trong những phần quan trọng nhất của grammar vì nó giúp diễn tả điều kiện, giả định và kết quả.
ZeroFirstSecondThird
Cách học: Xác định thời gian thật hay giả định rồi chọn đúng loại câu điều kiện.

Công thức 5 loại

  • Zero: If + present, present. (sự thật)
  • First: If + present, will + V. (có thể xảy ra)
  • Second: If + past, would + V. (giả định hiện tại)
  • Third: If + past perfect, would have + V3. (giả định quá khứ)
  • Mixed: If + past perfect, would + V. (quá khứ → hiện tại)

Second: If I had more time, I would learn Spanish.

Mixed: If I had studied medicine, I would be a doctor now.

Mini practice

Hoàn thành câu: If it rains, we ___ (stay) at home.

Đáp án mẫu: will stay (First conditional)

Các biến thể quan trọng

  • Unless = If not: Unless you study, you will fail.
  • Provided (that) = If (formal): You can go provided that you finish.
  • As long as = If: As long as you work hard, you will succeed.
  • In case = phòng khi: Take an umbrella in case it rains.
  • Dao ngu: Were I you, I would accept. / Had I known, I would have helped.

Ví dụ chuyển đổi giữa các loại

  • Real: If she studies hard, she will pass. (1st)
  • Unreal now: If she studied hard, she would pass. (2nd)
  • Unreal past: If she had studied hard, she would have passed. (3rd)
  • Mixed: If she had studied hard (past), she would be a doctor now (present).

Lỗi hay gặp

  • Dùng sai thì của mệnh đề if.
  • Nhầm second và third conditional.
  • Quên đảo cấu trúc khi dùng inversion.
  • Dùng will trong mệnh đề if ở first conditional.
  • Nhầm unless với if (unless = if NOT).

Vì sao module này quan trọng

  • Xuất hiện dày trong writing và speaking.
  • Rất quan trọng để diễn đạt giả định.
  • Là grammar cốt lõi cho trình độ trung cấp trở lên.
  • Hay gặp trong IELTS Speaking Part 3.
